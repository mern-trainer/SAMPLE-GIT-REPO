Hello World!

-> vs code -> text editor
-> git -> version control system
-> github -> cloud storage 

-> git init -> Initialize a local repository

-> git config --global user.name "You name" -> set your name
-> git config --global user.email "You email" -> set your email 

-> git add . -> add all files
-> git add <file_name_1> <file_name_2> -> add specific files 

-> git commit -m "your commit message" -> commit changes to the repository

-> git remote add <remote_name> <remote_url> -> add a remote repository
-> git remote remove <remote_name> -> remove a remote repository
-> git remote -> list remote repositories
-> git config --get remote.<remote_name>.url -> get the url of the remote repository

-> git push <remote_name> <branch_name> -> push changes to the remote repository
